# Feedback system

## Input

## Output

1. User ID, name
2. Scores: HW1~HW5
3. Midterm
4. Feedback: WER-sentence, WER-passage, Average WER, comments
5. Expected final grade range
6. Audio files to play using links

## File naming
+ UserID: F01~, M01~
+ Pre/Post
+ S(entence)/P(assage)
+ Song

e.g., F01-Pre-S01.wav, M01-Post-P.wav, M09-Song.wav
